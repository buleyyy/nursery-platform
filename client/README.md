# H. Ali Nursery — Client

React + Vite frontend untuk platform order management H. Ali Nursery.

## Development
```bash
npm install
npm run dev
```

## Environment Variables
Buat file `.env.local` untuk development lokal:
```
VITE_API_URL=http://localhost:3006
```

Untuk production, set di Vercel dashboard:
```
VITE_API_URL=https://nursery-platform-production.up.railway.app
```
